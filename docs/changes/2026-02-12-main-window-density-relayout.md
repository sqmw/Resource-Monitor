# 2026-02-12 Main Window Density Relayout

## Summary
- 解决主窗口顶部空白过大、组件间距过松的问题。
- 通过结构重排（主内容区 + 设置侧栏）从根上消除“右侧控件撑高头部”导致的大空洞。

## Root Cause
- 之前头部把“标题信息”和“大量控制组件”放在同一行容器中。
- 右侧控制区高度远高于左侧标题区，导致后续卡片区整体下移，形成大块空白。

## Changes
- 文件：`src/routes/+page.svelte`
- 结构重排：
  - 新增 `workspace` 双列布局：`main-content + control-stack`
  - 标题区仅保留业务信息，不再混合控制组件
  - 控制项独立为侧栏卡片容器
- 密度收敛：
  - 全局内容内边距下调
  - 卡片网格、进度区、预览区间距下调
  - meta 字体与标题区外边距收敛
- 响应式：
  - 小宽度下自动回退单列布局，设置栏置顶显示

## Result
- 首屏信息更紧凑，主要数据区域立即可见。
- 组件之间距离收敛，视觉密度提升。
- 不同分辨率下布局更稳定。
